# CHEAPESTFIN

> #### Using CheapShark's API to fetch data about game prices.
> #### Used Rest API to make GET requests to fetch data from cheapshark's servers.


<br>

> ## HOW TO USE IT?

```sh
cd cheapestfin
```
```
npm install
```
```
npm start
```
