# CHEAPESTFIN

### Small ReactJS app that finds cheapest price and other info for a game you search.
 
> Price data is fetched from CheapShark's open API.

> Used class based components.

> 


<br>

> ## HOW TO USE IT?

```sh
cd cheapestfin
```
```
npm install
```
```
npm start
```
