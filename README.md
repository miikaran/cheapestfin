# CHEAPESTFIN

### Small React app that finds cheapest price for a game you search.
 
>#### Built this to learn more about API's and how to use them in your applications.
>#### Used CheapShark's open API to fetch data about game prices.

<br>

> ## HOW TO USE IT?

```sh
cd cheapestfin
```
```
npm install
```
```
npm start
```
